# Statistical Data Analysis

Hands-on repo for the subject "Statistical Data Analysis" covering the full data analysis pipeline: data cleaning, EDA, clustering & dimensionality reduction, statistical modeling from raw, messy data to actionable insights.

## Topics covered

| # | Topic |
|---|-------|
| 1 | **Data Cleaning** |
| 2 | **Exploratory Data Analysis** |
| 3 | **Clustering & Dimensionality Reduction** |
| 4 | **Statistical Modeling** |
| 5 | **Hypothesis Testing & Inference** |

## Dataset

Dataset used in repo is a loan application data sourced from [OpenML](https://www.openml.org/search?type=data&status=active&qualities.NumberOfInstances=between_10000_100000&qualities.NumberOfFeatures=between_10_100&sort=qualities.NumberOfMissingValues&id=46458)
